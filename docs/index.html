色んな世界、あなただけの遊び場を見つけよう

どくさば！は、Minecraft Java Editionに対応するサーバーで、ユニークなゲームプレイが特徴です。

このサーバーには、PvPの世界、サバイバルの世界、経済の世界など、様々な世界が用意されており、新たにレシピやアイテムも追加されているので、あなた好みの遊び方を見つけましょう。

また、このサーバーは、Discordと連携しており、Discordでのチャットやボイスチャットも可能です。Discordに参加して、他のプレイヤーとコミュニケーションを取りながら、楽しい時間を過ごしましょう。

おすすめの遊び方

サバイバルワールドでの生活

サバイバルワールドでは、他のプレイヤーと協力して、資源を集め、建築を行い、生活を楽しむことができます。また、サバイバルワールドには、他のプレイヤーとのPvPが許可されているので、他のプレイヤーとの戦闘も楽しむことができます。

経済ワールドでの商売

経済ワールドでは、他のプレイヤーとの取引を行い、お金を稼ぐことができます。また、経済ワールドには、他のプレイヤーとのPvPが禁止されているので、安心して商売を行うことができます。

アスレチックワールドでの挑戦

アスレチックワールドでは、他のプレイヤーと競争を行い、アスレチックをクリアすることができます。また、アスレチックワールドには、他のプレイヤーとのPvPが禁止されているので、安心してアスレチックを楽しむことができます。

ミニゲームワールドでの対戦

ミニゲームワールドでは、他のプレイヤーと対戦を行い、勝利を目指すことができます。また、ミニゲームワールドには、他のプレイヤーとのPvPが許可されているので、他のプレイヤーとの戦闘も楽しむことができます。
